## Crear un Issue nuevo

### Vía Web (Más fácil)
1. URL: https://github.com/alvaroprzr/FIWARE-app/issues/new
2. **Título:** Issue con formato `Issue #<n>: <descripción>`
3. **Body:** Incluir objetivo, alcance y criterios de aceptación
4. Click "Submit new issue"

### Vía CLI
```bash
gh issue create \
  --title "Issue #24: Correcciones finales Stores Map, i18n, README y documentación" \
  --body "<descripción del alcance>" \
  --repo alvaroprzr/FIWARE-app
```
